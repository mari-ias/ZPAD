# Лабораторна робота №5  
## Наука про дані: веб-додаток для аналізу часових рядів

Цей проєкт реалізує інтерактивний веб-додаток для аналізу індексів стану рослинності (VCI, TCI, VHI) з використанням бібліотеки Streamlit.

## Функціональність

- Вибір типу індексу (VCI, TCI, VHI)
- Вибір області
- Фільтрація за роками та тижнями
- Сортування даних
- Відображення:
  - Таблиці
  - Часового графіка
  - Порівняння областей

## Контекст кібербезпеки

Методи аналізу часових рядів можуть застосовуватись для:
- Виявлення аномалій у мережевому трафіку
- Аналізу поведінкових патернів
- Моніторингу систем у реальному часі

## Запуск

```bash
streamlit run app.py
```
