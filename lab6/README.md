# Лабораторна робота №6
## Тема: Обробка відеопотоку в реальному часі з використанням OpenCV (C++)

Дана лабораторна робота присвячена розробці інтерактивної програми для обробки відеопотоку з веб-камери в реальному часі.

Програма реалізована мовою C++ з використанням бібліотеки OpenCV та дозволяє застосовувати різні алгоритми обробки зображень залежно від дій користувача (клавіатура, миша, слайдер).

Реалізовано модульну архітектуру, де кожен компонент системи винесений в окремий клас та файл.

## Опис програми

Ця програма реалізує інтерактивну обробку відеопотоку з камери в реальному часі з використанням OpenCV.
Користувач може змінювати режими обробки зображення, взаємодіяти з відео через клавіатуру та мишу, а також змінювати параметри обробки за допомогою GUI-елементів.
Після запуску відкриється вікно з відео з камери, де можна перемикати режими та взаємодіяти з зображенням у реальному часі.

## Етапи роботи
- Захоплення відео з камери (cv::VideoCapture)
- Відображення відеопотоку у вікні
- Реалізація обробки клавіатури (перемикання режимів)
- Реалізація обробки миші (малювання прямокутників)
- Реалізація різних режимів обробки зображення
- Додавання інтерактивних елементів (trackbar)
- Виведення службової інформації (FPS)
- Організація проєкту з використанням CMake

## Реалізовані режими
1 — Grayscale (перетворення у відтінки сірого)
2 — Canny edge detection (детекція границь)
3 — Gaussian blur (розмиття)
4 — Invert colors (інверсія кольорів)
5 — Sobel filter (градієнти зображення)
6 — Glitch effect (зсув кольорових каналів)
7 — Picture-in-picture (накладання зображення)

## Додаткові можливості
- Zoom (W / S)
- Rotation (A / D)
- Малювання прямокутників мишею
- Очищення малюнків (R)
- Регулювання яскравості (trackbar)
- Відображення FPS
- Робота в реальному часі

## Застереження
Програма використовує веб-камеру. Якщо камера зайнята іншою програмою (Zoom, Teams тощо), відеопотік може не відкриватися.

## Особливості запуску у WSL2
Для роботи з камерою у WSL2 необхідно:

1. Встановити usbipd-win на Windows
2. Підключити камеру:
```bash
usbipd list
usbipd attach --busid <ID>
```   
3. Запустити X-сервер (GWSL або VcXsrv)

Без цих налаштувань камера у WSL2 працювати не буде.
