# Звіт з лабораторної роботи №2
## Тема: Наука про дані: підготовчий етап

**Мета:** Ознайомитися з основами підготовки даних до аналізу: завантаженням через `urllib`, очищенням (data cleaning) за допомогою `Pandas`, фільтрацією та статистичним аналізом.

### Короткий опис завдань:

#### Частина 1: Аналіз VHI-індексу
* Налаштування віртуального середовища (`venv`).
* Автоматичне завантаження даних VHI для областей України з NOAA.
* Очищення даних, зміна індексів областей за українською абеткою.
* Створення функцій для фільтрації даних за роками та пошуку статистичних показників (min, max, середнє).

#### Частина 2: Аналіз енергоспоживання (Household Power Consumption)
* Робота з великим датасетом: очищення та вибірка за умовами (потужність > 5кВт, сила струму 19-20А тощо).
* Випадковий вибір 500 000 записів та аналіз груп споживання.
* Профілювання часу виконання операцій за допомогою `timeit`.
* Нормалізація даних та обчислення коефіцієнтів кореляції Пірсона та Спірмена.
