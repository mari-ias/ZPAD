# Лабораторна робота №4  
## Цифрова обробка та візуалізація сигналів

Дана лабораторна робота присвячена дослідженню гармонічних коливань, аналізу впливу адитивного білого гаусового шуму (AWGN) та реалізації методів цифрової фільтрації (фільтр Баттерворта) у середовищі Jupyter Notebook.

У роботі побудовано інтерактивну систему, що дозволяє в реальному часі маніпулювати параметрами фізичного сигналу та оцінювати ефективність алгоритмів очищення даних.

## Контекст кібербезпеки
Аналіз та фільтрація сигналів є критично важливими для фахівців із кібербезпеки:
* **Аналіз завад:** виявлення та нейтралізація Jamming-атак у каналах зв'язку.
* **Захист від Side-channel attacks:** аналіз побічних електромагнітних випромінювань від обладнання.
* **Моніторинг мережі:** виявлення аномальних сплесків у трафіку (DDoS) на фізичному рівні передачі даних.

## Етапи роботи

1. **Генерація сигналу:** Реалізація функції `harmonic_with_noise` для створення чистої гармоніки.
2. **Моделювання завад:** Накладання адитивного шуму з керованими параметрами середнього значення (`Mean`) та дисперсії (`Variance`).
3. **Цифрова обробка (DSP):** Реалізація та налаштування низькочастотного фільтра Баттерворта (Butterworth filter) 4-го порядку.
4. **Розробка UI:** Створення інтерактивного інтерфейсу за допомогою `ipywidgets` (слайдери, чекбокси, кнопки).
5. **Аналіз результатів:** Побудова динамічних графіків для порівняння зашумленого сигналу з відфільтрованим еталоном.

## Інструкція користувача

1. **Налаштування гармоніки:** Використовуйте слайдери `Amplitude`, `Frequency`, `Phase` для зміни форми сигналу.
2. **Керування шумом:** Змінюйте параметри `Mean` та `Variance`. Згідно з ТЗ, шум не перегенерується при зміні параметрів гармоніки.
3. **Фільтрація:** Налаштуйте частоту зрізу (`Cutoff`) та порядок фільтра (`Order`) для очищення сигналу.
4. **Reset:** Натисніть кнопку для миттєвого відновлення початкових параметрів.
